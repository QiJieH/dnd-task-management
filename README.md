这是一个基于react和令人惊叹的react-beautiful-dnd可拖拽库构建的网页程序。

## 本地部署 :whale:

项目使用create-react-app构建，你可以通过以下命令在本地运行本项目：

```shell
npm install
```

```shell
npm start
```

随后项目将会部署在 https://localhost:3000
